# Backbone Base
> A simple backbone application template.

### Installation
```sh
git clone https://github.com/ionutvmi/backbone-base.git myApp
cd myApp
npm install
```
To start developing: 

1. run `gulp develop`
2. visit `http://localhost:8080`

To build the project run `gulp build`.

### Project structure
```
+---assets
|   +---css

|   |   |   +---routers

\---test
    +---lib
    \---spec
```
